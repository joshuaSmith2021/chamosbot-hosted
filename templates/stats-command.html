!stats

!stats <minigame> <username(s)> -flags

  • Parameters
  • Flags
  • Examples

Parameters

The !stats command has two parameters: minigame, and username(s). Minigame is the minigame (ie skywars or bedwars) that the bot should get stats from. Username(s) is a space-separated list of usernames that the bot should fetch stats from.

Flags

With some minigames, you can specify a specific game mode to pull stats from. For bedwars, the format for one of these flags is -<number of teams>_<players per team> . In SkyWars, it would be -<team size>_<insane/normal (optional)> For example, the flag for Bedwars solo would be -eight_one , because there are eight teams in the game, with each team having one player. The flag for team insane skywars would be -team_insane .

Examples

For reference, here is a list of some example commands, along with a description with what exactly the command would result in.

  • poll !stats bw parcerx Returns Bedwars stats for parcerx.
  • poll !stats sw parcerx i41 Returns Skywars stats for parcerx and i41.
  • poll !stats bw parcerx -eight_one Returns solo Bedwars stats for parcerx.
  • poll !stats sw parcerx -team Returns team Skywars stats for parcerx.
  • poll !stats sw parcerx i41 -solo_insane Returns solo insane Skywars stats for parcerx.
