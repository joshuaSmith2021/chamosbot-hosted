Features

ChamosBot is a Discord bot focused on providing Hypixel stats to its users. Currently, it supports Bedwars, SkyWars, and the Pit, with Bedwars having the most features. ChamosBot also integrates with a web app that displays Bedwars stats over time.

  • Preview Stat Tracker

Technology

ChamosBot is written in Python using Discord.py, and is currently running on one of my Raspberry Pi 3 B+'s. I've thought about moving it to the cloud on an AWS virtual machine, but for the time being I'm keeping it on my simple machine because it doesn't serve very many servers. The data is fetched using the Hypixel API. When you add it to your Discord server, you provide it with your Hypixel API key.
If any of this sounds complicated, don't worry! This is just a bunch of nerd talk for other software developers. Setting up the bot is super simple.

Commands
  • poll !stats (bedwars | skywars | pit) (usernames) Sends a table with some stats from the given game for the given player(s). If multiple players are specified the best stats in each category are highlighted.
  • link !link (usernames) Sends a link to the web app for the given users.

Every command is handled by ChamosBot. If you have any issues, first make sure the bot is online. If the bot is offline, it is very possible that the Raspberry Pi ChamosBot runs on crashed or got disconnected from the network. In this case, I will know, and there isn't really a need for a bug report. If the bot is online and it is not responding to commands, sending strange responses, or misbehaving in any way, feel free to make a bug report at the GitHub page.
